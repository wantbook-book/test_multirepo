## Obtaining
### Breaking
Honeycomb blocks can be broken using any tool,[1] and always drop as an item.

| Block    | Honeycomb Block     |
|----------|---------------------|
| Hardness | 0.6                 |
|          | Breakingtime (secs) |
| Default  | 0.9                 |

## Usage
### Note Blocks
Honeycomb blocks can be placed under note blocks to produce flute sounds. ‌[Bedrock Edition  only][2]
