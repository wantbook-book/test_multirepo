## Obtaining
### Breaking
Granite slabs can be mined using any pickaxe.

| Block     | Granite Slab          |
|-----------|-----------------------|
| Hardness  | 1.5                   |
| Tool      |                       |
|           | Breakingtime (sec)[A] |
| Default   | 7.5                   |
| Wooden    | 1.15                  |
| Stone     | 0.6                   |
| Iron      | 0.4                   |
| Diamond   | 0.3                   |
| Netherite | 0.25                  |
| Golden    | 0.2                   |


↑ Times are for unenchanted tools as used by players with no status effects, measured in seconds. For more information, see Breaking Speed.

## Usage
For information about the placement mechanics of all slabs, see Slab § Usage.

Granite slabs can be placed under note blocks to produce "bass drum" sounds.
