# Language
Languages is a feature that allows changing languages for people who prefer to play in a language other than their default setting. The language menu is accessible via a button in the options/settings menu below general.

Minecraft's default font Mojangles is made up of a grid of dots. Each character supported by the font is between 1 and 6 dots wide. All ASCII characters between 32 (space) and 126 (~) are five dots wide, with the following exceptions:
