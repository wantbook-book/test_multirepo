# Tutorials/Loading a resource pack
These tutorials are meant to help you use resource packs in Minecraft. Resource packs allow players to customize textures, music, sounds, language files, end credits, splashes, and fonts but do not change the actual gameplay.

## Contents
- 1 Any platform
- 2 Direct methods
	- 2.1 Windows (Java Edition)
	- 2.2 MacOS
	- 2.3 Linux

## Any platform
1. Download a resource pack, you should get a .zip file
2. Open the folder to which you downloaded the file
3. Copy the .zip file
4. Open Minecraft
5. In Minecraft, click on "Options", then "Resource Packs"
6. Choose "Open resource pack folder"
7. Paste the file into the resource folder in Minecraft
8. Go back to Minecraft, the resource pack should now be listed (if not listed then go to the main menu by pressingEscand repeat step 5) and you can activate it by choosing it and hitting "Done". The game will now reload itself.

## Direct methods
### 
1. Download the resource pack you want
2. Copy the file you downloaded
3. PressWin+R. A window titled "Run" should appear
4. In the textbox inside the "Run" window, type "%appdata%" and click "Ok"
5. In the folder that just appeared, open the ".minecraft" folder
6. In the ".minecraft" folder, open the "resourcepacks" folder
7. Paste the resource pack in this folder
8. Open Minecraft, select "Options", then "Resource Packs", select your resource pack and select "done"
9. You can skip 4,5, and 6 if you want to, you have to type this in the "Run" window: %appdata%/.minecraft/resourcepacks

### MacOS
1. Download a resource pack of your choice
2. Do NOT unzip the resource pack just move the .zip file to Minecraft's resource pack folder (Should be at~/Library/Application Support/minecraft/resourcepacks)
3. Open up Minecraft and login to your account
4. At the main menu select "Options", then "Resource Packs"
5. Now select your new resource pack and then select done

To access ~/Library/ on a Mac running MacOS 10.7 and above, switch to Finder, open the 'Go' Menu, and while holding down Option, select 'Library'

### Linux
1. Download a resource pack, you should get a .zip file
2. Open the folder to which you downloaded the file
3. Copy the .zip file
4. Type~/.minecraft/resourcepacks/in your file manager's address bar or in the console (If it is not showing up, press the three dots in the corner and make sure that the "show hidden files" box has a check next to it, then repeat step 4.)
5. Paste the file into the directory
6. Open Minecraft
7. Go to "Options", then "Resource Packs" The resource pack should now be listed and you can activate it by choosing it and hitting "Done".
