# Vines (feature)
Vines is a feature that consist of a single vines block.

## Contents
- 1 Generation
- 2 Data values
	- 2.1 ID
	- 2.2 Config
- 3 History

## Generation
Vines are features made of a single vines block. They can be found in the Jungle, Sparse Jungle, Bamboo Jungle and Lush Caves biomes.
