## Obtaining
### Breaking
Granite walls can be mined using any pickaxe. If mined without a pickaxe, they drop nothing.

| Block     | Granite Wall          |
|-----------|-----------------------|
| Hardness  | 1.5                   |
| Tool      |                       |
|           | Breakingtime (sec)[A] |
| Default   | 7.5                   |
| Wooden    | 1.15                  |
| Stone     | 0.6                   |
| Iron      | 0.4                   |
| Diamond   | 0.3                   |
| Netherite | 0.25                  |
| Golden    | 0.2                   |

1. ↑Times are for unenchanted tools as used by players with no status effects, measured in seconds. For more information, see Breaking Speed.

### Natural generation
An isometric render of a desert village armorer building.
Each desert village armorer building generates with 2 granite walls as the top part of its chimney.
